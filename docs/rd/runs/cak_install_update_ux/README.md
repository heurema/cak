# CAK Install / Update UX

Status: decision-ready for v0 direction; not implementation-ready.

This research packet decides the distribution contract for the public `cak`
CLI and its host adapters.

The packet is intentionally scoped to local developer use and CI. It does not
decide public package-manager distribution, auto-update daemons, remote policy
marketplaces, or host-specific runtime interception.
