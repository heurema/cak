# RDR-001 — What is an agent-native skill?

This directory contains the exploratory research-run packet for RDR-001.

This is not the final RDR and is not decision-ready. It uses
`docs/rd/03_skill_architecture_synthesis.md` as input and tests the working
hypothesis that agent-native skills may be compiled, governed bridges between
Evidence / Provenance IR and Runtime / Control IR.

The intended next step is evidence audit plus structured debate before drafting
the final RDR.
